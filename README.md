# Simple Calculator
A beginner-friendly calculator app built with Vite, vanilla JavaScript, and Tailwind CSS. It helps reinforce core programming concepts like variables, data types, operators, functions, and DOM manipulation.
## Features
- Perform basic arithmetic: Add, Subtract, Multiply, Divide
- Reset button to clear input and result
- Dynamic UI updates using DOM
- Modular JavaScript structure (`main.js` + `calculator.js`)
- Styled with Tailwind CSS
## Tech Stack
- Vite
- Vanilla JavaScript (ES Modules)
- Tailwind CSS
